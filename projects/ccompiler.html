C Compiler

Overview

A compiler from C90 for MIPS IV architecture. Implemented in C++. The lexer is built using GNU Flex and the parser using GNU Bison. The code for the compiler can be found at https://github.com/nikonikolov/c_compiler. The project was built as part of our coursework on Language Processors.

Outcome

One of the hardest projects I have done. Got me to have huge respect for gcc, clang and llvm.

I will avoid implementation details - you can check them on GitHub. For the 2 months we had to complete the coursework (during term time), I got to fully implement lots of C features: all expressions, loops, conditionals, arrays, pointers, functions, linkabe assembly code. It was very fun playing with MIPS ABI and grammar theory. I liked the coursework a lot as it required using different complex algorithms, choosing data structures that balance between memory and speed and generating assembly.
