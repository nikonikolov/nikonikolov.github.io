Pepper Recognition and Speech

Overview

This project was built during the UK NAO Hackathon in June 2016. The topic of the hackathon was to build useful additional functionality for a Pepper robot. This is a robot developed by SoftBank Robotics Corp. The idea of our team was to enable Pepper help people learn new language. For this purpose, after showing an object to Pepper and giving him a voice command, he recognized the object by taking a picture of it and pronouncing it in another language.

Outcome

It was very fun playing with Pepper. I ended up implementing a lot of embedded sofware and adding on Pepper's firmware. For the implementation we used Clarifai API and Yandex text-to-speech engine. It turned out that these robots are currently hard to interface with, so there is definitely some cool work to be done in this area.
